# Release exclusions

The following items are intentionally excluded from this curated submission package:

- Superseded non-grouped TMD benchmark outputs and scripts.
- Temporary smoke-test outputs and intermediate logs.
- Raw AlphaFold PDB files, which are retrievable from their public source.
- Raw sequencing reads, which remain available through the public GEO accession GSE128578.
- The large ESM2-650M embedding file when it exceeds repository size limits; the repository retains the extraction workflow and benchmark outputs.

These exclusions do not remove any canonical script or result required to interpret or reproduce the analyses reported in the manuscript and Supporting Information.
